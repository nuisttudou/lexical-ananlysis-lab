# Mycompiler

## Run

 python mycompiler.py -s [file] [options]

## Development environment

Ubuntu 18.04 

Python 3.6.9

vscode 1.44.2

## Options
​    -h, --help         show help
​    -s file         select source file
​    -l              lexical-ananlysis

## Examples
​    python mycompiler.py -h
​    python mycompiler.py -s source.test -a
